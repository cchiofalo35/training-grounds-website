# Training Grounds — Martial Arts Website

This is the website for Training Grounds, a martial arts gym in St Leonards, Sydney. It is a static HTML site hosted on GitHub Pages.

## Site Structure

- index.html — Main landing/home page
- coaches.html — Coaching team and philosophy
- kids.html — Kids academy programs
- womens.html — Women's self-defence and First Move program
- timetable.html — Class timetable
- join.html — Membership plans and signup (Clubworx iframe)
- contact.html — Contact info and location map
- sitemap.xml — SEO sitemap
- robots.txt — Crawler config
- CNAME — Custom domain config

## Design System

- Colors: Charcoal #1E1E1E, Warm Accent #C9A87C, Off-White #FAFAF8, Dark BG #121212
- Fonts: Bebas Neue (headings), Inter (body)
- Nav: Dark fixed navbar with backdrop-filter blur
- All images hosted on Squarespace CDN (do not change image URLs unless specifically requested)

## Guidelines for Making Changes

1. Only change what is requested — Do not redesign or restructure anything unless specifically asked.
2. Preserve the existing design — Keep all styling, colors, fonts, and layout consistent with the design system above.
3. Be careful with HTML — This is a static site with no build step. All changes are directly to HTML files.
4. Keep it professional — This is a martial arts gym's website.
5. Do not change image URLs — Images are served from Squarespace CDN. Do not replace or modify image sources.
6. Do not change the Clubworx iframe — The signup form on join.html is a Clubworx integration. Do not modify it.
7. When in doubt, ask — If a request is ambiguous, comment on the issue asking for clarification.
